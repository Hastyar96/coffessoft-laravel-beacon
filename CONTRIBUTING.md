# Contributing to Laravel Beacon

## Development Setup

```bash
git clone https://github.com/Hastyar96/coffesoft-laravel-beacon.git
cd coffesoft-laravel-beacon
composer install
```

## Running Tests

```bash
vendor/bin/phpunit
```

## Code Style

This package follows PSR-12 coding standards.

## Pull Request Process

1. Ensure tests pass before submitting
2. Update documentation if adding new features
3. Update CHANGELOG.md with any changes
