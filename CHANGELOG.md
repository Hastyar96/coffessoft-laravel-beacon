# Changelog

## v1.0.0 (2024-07-04)

Initial stable release of Laravel Beacon.

- Project scanning (Models, Controllers, Routes, Services, Repositories, and more)
- Intelligence engines (Architecture Detection, Security Analysis, Performance Analysis, etc.)
- Multiple export formats (JSON, Markdown)
- AI-powered context generation for development tasks
- Code quality review with severity ranking
- Route health analysis
- Refactoring plan generation
- Incremental scan caching
- Laravel 9.x, 10.x, 11.x, 12.x, 13.x support
- PHP 8.1+ support
